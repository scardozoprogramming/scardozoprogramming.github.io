Project Overview

Interactive Training and Match Tools

The project combines a visual training guide for beach volleyball 2v2 with practical tournament and scoreboard tools. It focuses on turning tactical knowledge into an interactive web experience.

2 Core pages
5 Language options

Scope

What I Built

  • Animated training systems Interactive court sequences for blocking signals, defense roles and setting calls.
  • Movement and call references Structured callbook, movement phases and tactical descriptions for players.
  • Tournament organizer Team setup, group generation, standings and knockout bracket simulation.
  • Match scoreboard Live score controls with side-change reminders every seven total points.

Case Study

Problem, Solution, Result

  • Problem Beach volleyball tactics are difficult to explain with static notes because positioning changes quickly during each rally.
  • Solution I built an interactive web app with animated court diagrams, language support, tournament generation and scoreboard behavior.
  • Result The project became a self-contained training and match-management tool that can be opened directly in the browser.

Technical Work

Responsive UI

Multilingual content
