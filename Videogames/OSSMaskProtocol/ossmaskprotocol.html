OSS MASK PROTOCOL

Programmer & Producer — 48h Game Jam — Unity — Team of 8

Project Overview

OSS Mask Protocol is a Unity game developed during a 48-hour game jam. The project was built by a multidisciplinary team of 8 members, combining fast prototyping, production coordination and gameplay system implementation under a strict deadline.

My Role — Programmer & Producer

I worked as both Programmer and Producer, coordinating development while implementing core gameplay systems. My work focused on player input, interaction logic, NPC behavior support, task prioritization and final delivery.

Programming

  • Implemented the player input system using Unity Input System.
  • Developed core player interaction logic.
  • Collaborated on the NPC behavior system.
  • Helped integrate gameplay features inside Unity under game jam constraints.

Production & Delivery

  • Coordinated art, music, modeling and animation workflow.
  • Managed task prioritization under a 48-hour deadline.
  • Produced the final presentation and gameplay video.
  • Prepared the final build for submission.

Core Gameplay & Jam Focus

The project focused on quickly delivering a playable game loop with clear player interaction, NPC behavior and a polished enough build for presentation within the 48-hour production window.
