Game Overview

BOO! I'm Inside Your Walls is a stealth puzzle game where players control Bunyo, a small ghost working for a scare company in order to earn enough money to reach the afterlife. Players must infiltrate houses, possess objects, create illusions and terrify residents before a ghost hunter arrives.

My Role — Gameplay Programmer & Game Designer

I mainly worked as Gameplay Programmer, Game Designer and Unity Integration Support, focusing on player abilities, gameplay systems, level layouts and scene organization inside Unity. I also collaborated on early procedural room generation prototypes before transitioning into handcrafted level design.

Gameplay Programming

  • Programmed most of the player's gameplay abilities including Super Speed, Teleportation, NPC X-Ray Vision, Telepathy and Invisibility.
  • Developed gameplay logic, cooldown systems, input handling and gameplay state integration.
  • Worked on player interaction systems and gameplay ability implementation inside Unity.
  • Collaborated on early procedural room generation prototypes before switching to manual level layouts.

Game Design & Unity Integration

  • Designed several house layouts and room distributions for different levels.
  • Helped define gameplay progression and player ability distribution across levels.
  • Integrated prefabs, textures and environment assets inside Unity scenes.
  • Organized scene structures and adjusted Rigidbody setups, prefab scales and physics consistency.

Core Gameplay

The game focuses on short and intense stealth sessions lasting less than 90 seconds. Players must strategically use possessions, illusions and special abilities to increase residents' fear levels without being detected.

Core Loop

Infiltrate the house, observe residents, possess objects, create illusions, raise fear levels and escape before the ghost hunter arrives.

Art Direction

The visual style takes inspiration from Tim Burton aesthetics, using exaggerated character proportions, twisted object shapes and a combination of hand-painted textures with PBR materials. The red-and-black visual identity reinforces the game's eerie but playful atmosphere.
