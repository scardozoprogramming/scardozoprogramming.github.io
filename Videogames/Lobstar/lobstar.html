Game Overview

LOBSTAR! is a fast-paced arcade action game developed in Unity. Players control a kaiju with one goal: to destroy the world. The game combines beat ’em up combat, arcade-style pacing and chaotic destruction-focused gameplay.

My Role — Game Programmer

I worked mainly as a Game Programmer, focusing on the early-game experience, UI flow, camera behavior, player feedback and presentation systems. My work included title screen animation, menu interactions, intro sequence implementation, camera transitions, sound feedback and custom typography integration.

UI & Presentation Programming

  • Designed and implemented the game’s custom typography for the title and UI presentation.
  • Programmed the title screen movement, animation behavior and menu button interactions.
  • Implemented the game introduction flow leading into the comic sequence.

Camera, Audio & Feedback

  • Worked with another programmer on the camera system, including map view behavior and zoom transitions.
  • Integrated sound effects to improve player feedback and overall game feel.
  • Helped polish the presentation layer to make the game feel more responsive and arcade-like.

Core Gameplay

LOBSTAR! is a fast-paced arcade action game where players control a destructive kaiju rampaging through the city. Gameplay focuses on responsive movement, beat ’em up combat, environmental destruction and exaggerated player feedback to create a chaotic and satisfying gameplay experience.
